Классификация ошибок
Код ошибки Описание ошибки Случаи возникновения ошибки
101 Данная ошибка не критична. При выводе данной ошибки новая строка в таблицу не добавляется. Возникает при неправильном введении данных в поля для ввода, либо при отсутвии данных в полях.
102 Данная ошибка не критична. При выводе данной ошибки строка не удаляется из таблицы. Возникает тогда, когда не выбрана ни одна из строк.
103 Данная ошибка не критична. При выводе данной ошибки строка в таблице не изменяется. Возникает при неправильном введении данных в поля для ввода, либо при отсутвии данных в полях.Так же может возникнуть если не выбрана ни одна из строк в таблице.
104 Данная ошибка не критична. При выводе данной ошибки таблица не экпортируется в Microsoft Office или Microsoft Exile. Возникает тогда, когда не выведена ни одна из таблиц, или когда пользователь не сохраняет файл с уже выведеной в него таблицей.
105 Данная ошибка не критична. При выводе данной ошибки авторизация в ПО не осуществляется. Возникает тогда, когда не указан ни пароль, ни логин, либо пароль и логин указан неверно, так же может быть неверно указан только пароль или же только логин.
*Возможно таблица будет пополняться новыми записями
